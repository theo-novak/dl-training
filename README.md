# Quant Finance Training
A home for learning on financial data. Using artificial data for now since there is not much to learn from the price of a stock (remember from binomial: it either goes up or down) but alas, might be useful if you simply switch your bar to something more nuanced.

This started as a home for deep learning (hence the name dl-training), but I expanded it to include more topics in quant finance for those with little knowledge on the topic. I will continue adding notebooks as I write them to include more material and cover a wider area of topics, time given 🤞.
